## Using Rider

1. Open your project in Rider.
2. Right-click on the solution in Solution Explorer.
3. Click **"Manage NuGet Packages"**.

   ![image](https://github.com/user-attachments/assets/603c0606-bb15-4308-8c4b-2a475894f2e6)

4. You'll see the package list at the bottom of your IDE:

   ![image](https://github.com/user-attachments/assets/7dd58e57-c1c7-463e-aad3-3d3a633cc8c3)

5. Update packages:
   - **Individual packages**: Click the update icon next to each package

     ![image](https://github.com/user-attachments/assets/8dc20ba9-56f9-4d70-8170-df11a0979f33)

   - **All at once**: Click the update all button

     ![image](https://github.com/user-attachments/assets/0caf040f-cc3b-42bf-9edf-2d18ef4b8760)
