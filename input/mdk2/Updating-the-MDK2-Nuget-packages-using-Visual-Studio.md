> [!TIP]
> **Using MDK Hub?** The Hub can update packages automatically. See the [Updates & Maintenance](index.html#updates--maintenance) section for details.

## Using Visual Studio

1. Open your project in Visual Studio.
2. Right-click on the solution in Solution Explorer.
3. Click **"Manage NuGet Packages for Solution..."**

   ![image](https://github.com/user-attachments/assets/30386329-43aa-4510-ab19-f1d0ae3802b9)

4. Click the **"Updates"** tab to see outdated packages.

   ![image](https://github.com/user-attachments/assets/85504d2e-bebe-4f3e-ba47-7c39f30e8b0b)

5. Select the MDK packages you want to update and click **"Update"**.

   ![image](https://github.com/user-attachments/assets/1e7d30c2-f7f3-4e55-b08c-105c2183bed2)

6. Click **"Apply"** to complete the update.
