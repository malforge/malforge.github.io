## Summary

```csharp
public decimal ToDecimal(decimal defaultValue)
```

Attempts to get this value as a [decimal](https://docs.microsoft.com/en-us/dotnet/api/system.decimal?view=netframework-4.6) . If the value is empty or cannot be understood as this data type, the defaultValue will be returned instead.

## Returns

[decimal](https://docs.microsoft.com/en-us/dotnet/api/System.Decimal?view=netframework-4.6)



## Parameters

* [decimal](https://docs.microsoft.com/en-us/dotnet/api/System.Decimal?view=netframework-4.6) defaultValue
