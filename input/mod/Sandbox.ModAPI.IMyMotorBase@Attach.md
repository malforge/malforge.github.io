## Summary

```csharp
public void Attach(IMyMotorRotor rotor, bool updateGroup)
```

Attaches a specified nearby rotor/wheel to the stator/suspension block

## Parameters

* [IMyMotorRotor](Sandbox.ModAPI.IMyMotorRotor) rotor
* [bool](https://docs.microsoft.com/en-us/dotnet/api/System.Boolean?view=netframework-4.6) updateGroup
## Remarks

The rotor to attach must already be in position before calling this method.
