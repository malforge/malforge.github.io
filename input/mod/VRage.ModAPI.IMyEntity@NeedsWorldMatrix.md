## Summary

```csharp
public bool NeedsWorldMatrix { get; set; }
```

Gets or sets if [WorldMatrix](VRage.ModAPI.IMyEntity@WorldMatrix) should be calculated when parent WorldMatrix is changed.

## Returns

[bool](https://docs.microsoft.com/en-us/dotnet/api/System.Boolean?view=netframework-4.6)

## Remarks

Enabling it on big amount entities may lower simulation speed
