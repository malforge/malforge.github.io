## Summary

```csharp
public bool IsPositionInNaturalGravity(Vector3D position, double sphereSize)
```

This quickly checks if a given position is in any natural gravity.

## Returns

[bool](https://docs.microsoft.com/en-us/dotnet/api/System.Boolean?view=netframework-4.6)

True if there is natural gravity at this position, false otherwise.

## Parameters

* [Vector3D](VRageMath.Vector3D) position
* [double](https://docs.microsoft.com/en-us/dotnet/api/System.Double?view=netframework-4.6) sphereSize
