## Summary

```csharp
public static float UniformFloatFromSeed(int seed)
```

Compute a float in the range [0, 1) created from the the seed. For uniformly distributed seeds this method will produce uniformly distributed values.

## Returns

[float](https://docs.microsoft.com/en-us/dotnet/api/System.Single?view=netframework-4.6)

A float in the range [0, 1)

## Parameters

* [int](https://docs.microsoft.com/en-us/dotnet/api/System.Int32?view=netframework-4.6) seed
