## Summary

```csharp
public void CreateModelPhysics(PhysicsSettings settings)
```

Creates physical body when [DetectorColliderCallback](VRage.ModAPI.PhysicsSettings@DetectorColliderCallback) is null, or physics detector, when [DetectorColliderCallback](VRage.ModAPI.PhysicsSettings@DetectorColliderCallback) is not null. Created physics has a collision model, taken from entity model

## Parameters

* [PhysicsSettings](VRage.ModAPI.PhysicsSettings) settings
