## Summary

```csharp
public short ToInt16(short defaultValue)
```

Attempts to get this value as a [short](https://docs.microsoft.com/en-us/dotnet/api/system.int16?view=netframework-4.6) . If the value is empty or cannot be understood as this data type, the defaultValue will be returned instead.
