## Summary

```csharp
public void Attach(IMyPistonTop top, bool updateGroup)
```

Attaches a specified nearby top part to the piston block

## Parameters

* [IMyPistonTop](Sandbox.ModAPI.IMyPistonTop) top
* [bool](https://docs.microsoft.com/en-us/dotnet/api/System.Boolean?view=netframework-4.6) updateGroup
## Remarks

The top to attach must already be in position before calling this method.
