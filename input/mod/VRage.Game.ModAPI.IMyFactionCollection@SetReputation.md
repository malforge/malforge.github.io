## Summary

```csharp
public void SetReputation(long fromFactionId, long toFactionId, int reputation)
```

Sets reputation between 2 factions Note: Faction 1 always has same reputation to Faction 2, as Faction 2 to Faction 1

## Parameters

* [long](https://docs.microsoft.com/en-us/dotnet/api/System.Int64?view=netframework-4.6) fromFactionId
* [long](https://docs.microsoft.com/en-us/dotnet/api/System.Int64?view=netframework-4.6) toFactionId
* [int](https://docs.microsoft.com/en-us/dotnet/api/System.Int32?view=netframework-4.6) reputation
