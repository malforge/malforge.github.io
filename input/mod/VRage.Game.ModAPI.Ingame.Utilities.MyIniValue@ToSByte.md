## Summary

```csharp
public sbyte ToSByte(sbyte defaultValue)
```

Attempts to get this value as a [sbyte](https://docs.microsoft.com/en-us/dotnet/api/system.sbyte?view=netframework-4.6) . If the value is empty or cannot be understood as this data type, the defaultValue will be returned instead.
