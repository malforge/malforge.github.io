## Summary

```csharp
public void CreateSpherePhysics(PhysicsSettings settings, float radius)
```

Creates physical body when [DetectorColliderCallback](VRage.ModAPI.PhysicsSettings@DetectorColliderCallback) is null, or physics detector, when [DetectorColliderCallback](VRage.ModAPI.PhysicsSettings@DetectorColliderCallback) is not null. Created physics has a sphere collision

## Parameters

* [PhysicsSettings](VRage.ModAPI.PhysicsSettings) settings
* [float](https://docs.microsoft.com/en-us/dotnet/api/System.Single?view=netframework-4.6) radius
