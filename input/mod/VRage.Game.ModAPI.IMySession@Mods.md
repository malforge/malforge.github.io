## Summary

```csharp
public List<ModItem> Mods { get; set; }
```

Gets or sets mods added to current game

## Returns

[List<ModItem>](https://docs.microsoft.com/en-us/dotnet/api/System.Collections.Generic.List-1?view=netframework-4.6)

## Remarks

Setting mods won't add them to current game, but will save them in game save
