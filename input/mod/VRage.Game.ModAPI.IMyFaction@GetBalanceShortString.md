## Summary

```csharp
public string GetBalanceShortString()
```

Gets balance of an account associated with faction. Format is 'BALANCE CURRENCYSHORTNAME'.

## Returns

[string](https://docs.microsoft.com/en-us/dotnet/api/System.String?view=netframework-4.6)

Current balance of the account in form of formatted string. If Banking System does not exist method returns null.
