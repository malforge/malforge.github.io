## Summary

```csharp
public void RegisterDestroyHandler(int priority, Action<object, MyDamageInformation> handler)
```

Registers a handler for when an object in game is destroyed.

## Parameters

* [int](https://docs.microsoft.com/en-us/dotnet/api/System.Int32?view=netframework-4.6) priority
* [Action<object, MyDamageInformation>](https://docs.microsoft.com/en-us/dotnet/api/System.Action-2?view=netframework-4.6) handler
