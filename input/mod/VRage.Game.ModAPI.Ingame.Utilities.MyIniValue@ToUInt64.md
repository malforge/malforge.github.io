## Summary

```csharp
public ulong ToUInt64(ulong defaultValue)
```

Attempts to get this value as a [ulong](https://docs.microsoft.com/en-us/dotnet/api/system.uint64?view=netframework-4.6) . If the value is empty or cannot be understood as this data type, the defaultValue will be returned instead.

## Returns

[ulong](https://docs.microsoft.com/en-us/dotnet/api/System.UInt64?view=netframework-4.6)



## Parameters

* [ulong](https://docs.microsoft.com/en-us/dotnet/api/System.UInt64?view=netframework-4.6) defaultValue
