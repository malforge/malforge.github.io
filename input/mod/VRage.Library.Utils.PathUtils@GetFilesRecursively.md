## Summary

```csharp
public static String[] GetFilesRecursively(string path, string searchPattern)
```

Gets Files Recursively from specified path with specified search pattern. Method gives access only to sub-folders of User Data Path or Mods Path.

## Returns

String[] <sub>prohibited</sub>

array of names of the files

## Parameters

* [string](https://docs.microsoft.com/en-us/dotnet/api/System.String?view=netframework-4.6) path
* [string](https://docs.microsoft.com/en-us/dotnet/api/System.String?view=netframework-4.6) searchPattern
