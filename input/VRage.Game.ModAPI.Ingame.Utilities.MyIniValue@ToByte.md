## Summary

```csharp
public byte ToByte(byte defaultValue)
```

Attempts to get this value as a [byte](https://docs.microsoft.com/en-us/dotnet/api/system.byte?view=netframework-4.6) . If the value is empty or cannot be understood as this data type, the defaultValue will be returned instead.

## Returns

[byte](https://docs.microsoft.com/en-us/dotnet/api/System.Byte?view=netframework-4.6)



## Parameters

* [byte](https://docs.microsoft.com/en-us/dotnet/api/System.Byte?view=netframework-4.6) defaultValue
