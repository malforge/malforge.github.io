public interface IMyContractGridHauling: IMyContract

Describes grid hauling contract where player physically transports a cargo grid. Either provide TargetGridId for an existing grid, or PrefabName and GridPosition to have the contract spawn one.

Properties

Member Description
GridPosition The position where the cargo grid is or will be spawned.
PrefabName Prefab name to spawn. Ignored when TargetGridId points to an existing grid.
RiskLevel Risk level of the contract (0=Low, 1=Moderate, 2=High)
TargetGridId Entity id of an existing grid to be hauled. Set to 0 to spawn from PrefabName instead.
Collateral Gets money that player would loose if he won't complete contract
