MyContractGridHauling

Assembly: Sandbox.Game.dll

public class MyContractGridHauling: IMyContractGridHauling, IMyContract

Implements IMyContractGridHauling interface so you can create your own grid hauling contract. Call IMyContractSystem.AddContract with this instance. Either set TargetGridId for an existing grid, or PrefabName and GridPosition to have the contract spawn one.
