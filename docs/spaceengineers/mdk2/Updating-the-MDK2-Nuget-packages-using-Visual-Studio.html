Platform Support

  • Running MDK on Linux (functional, but unsupported)

After opening your project, right-click on the solution node in Solution Explorer, then find and click
Manage Nuget Packages for Solution...

This opens the package manager console. Click on the Updates tab to see any packages that needs updates.

Here we can see that all the MDK packages are out of date and needs to be updated. Select them all and click Update.

Hit Apply and your packages will be updated.
